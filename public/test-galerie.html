📝 Pour l'Administrateur :

Pour ajouter des images à la galerie publique :

  1. Connectez-vous au dashboard admin
  2. Allez dans Admin → Galeries
  3. Ajoutez de nouvelles images
  4. Cochez "Active" pour les rendre publiques
  5. Cochez "Mise en avant" pour les afficher en vedette
